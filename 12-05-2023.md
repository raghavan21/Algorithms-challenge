**Algorithm:**  
                        
1. The given codebook represents each character in the original string with a unique bit pattern.
2. The encoded string is a sequence of bits that represent the characters in the original string using the bit patterns from the codebook.
3. To decode the encoded string, we need to start at the beginning of the string and keep reading the bits until we find a bit pattern that matches one of the characters in the codebook.
4. Once we find a match, we add the corresponding character to our decoded string and move on to the next set of bits.
5. We repeat this process until we have processed all the bits in the encoded string and produced the full decoded string.
6. In the given example, the encoded string is a long sequence of bits that represent a phrase. To decode the phrase, we need to use the given codebook to find the characters corresponding to each set of bits in the encoded string.
7. We start at the beginning of the encoded string and read the first few bits "111110". This matches the bit pattern for the character "t" in the codebook, so we add "t" to our decoded string.
8. We continue reading the encoded string and decoding the characters until we reach the end of the string, producing the final decoded phrase

from typing import Dict

def decode(encoded: str, codebook: Dict[str, str]) -> str:
    decoded = ""
    current_code = ""
    for bit in encoded:
        current_code += bit
        for symbol, code in codebook.items():
            if current_code == code:
                decoded += symbol
                current_code = ""
                break
        else:
            continue
    if current_code:
        raise ValueError("Invalid encoding")
    return decoded
